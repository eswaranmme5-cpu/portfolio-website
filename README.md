# 🌐 Personal Portfolio Website

This is a simple personal portfolio website built using **HTML**.  
It contains two pages: a homepage and a contact page.

## 📄 Pages

### 🏠 index.html
The **homepage** of the website.  
It introduces the portfolio and may include:
- About section
- Skills or projects
- Basic personal information
- Navigation to the contact page

### 📞 contact.html
The **contact page** where visitors can find ways to connect with the owner.  
It may include:
- Contact information
- Email or social links
- Simple contact form

## 🛠 Technologies Used
- HTML5
